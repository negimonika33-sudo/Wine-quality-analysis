#Wine Quality Data Analysis

An Exploratory Data Analysis (EDA) project focusing on investigating the chemical properties of wine and how they influence overall quality ratings.

## 📊 Key Insights Found
* *Volatile Acidity:* Higher levels of volatile acidity tend to correlate with lower quality scores, as it can lead to an unpleasant, vinegar-like taste.
* *Distribution Trends:* Many chemical elements like volatile acidity show a right-skewed distribution, requiring specific visualizations to analyze properly.

## 🛠️ Tech Stack & Libraries
* *Language:* Python
* *Data Manipulation:* Pandas, NumPy
* *Data Visualization:* Matplotlib, Seaborn
